# Example: landing page

Neo-Minimal Permaweb patterns for marketing or editorial surfaces. Align with `shadcn/ui` + **Luma** sensibility (polish, restraint), then apply this system’s overrides (no radius, no shadows except tooltip edge cases, border-led layout).

## Hero

- Headline: sentence case, DM Sans 500, no all-caps eyebrow unless the brand explicitly requires it.
- CTA pair: default button = white or neutral border; one accent action only if the page commits to a single accent family—no filled primary chroma by default.
- Avoid hero “card” on background: use flat bands separated by rules; prefer border separation over large gaps between related blocks.

## Sections

- Alternate **flat sections** with top/bottom borders rather than boxed cards; connected borders, no doubled edges where sections meet.
- Feature lists: icon optional and minimal; align to a simple grid; separators between rows if the list is long.

## Trust and footer

- Logo row: monochrome or single accent; align to baseline grid. Use `three.js` for a custom mark only if identity calls for it—keep motion structural, not flashy.
- Footer: border-top, small 400 text, link styles are underline or color shift—not button chrome.
